# Yerushalmi Bava Kamma

Prototype interactive study site for **Talmud Yerushalmi Bava Kamma**, modeled after the existing Daf Yomi visual guide style but kept in a completely separate repo.

Current prototype:

- Masechet: Yerushalmi Bava Kamma
- Daf: ב
- Page: `dafim/bava-kamma/2/index.html`

## Local preview

```bash
python3 -m http.server 8787
# open http://localhost:8787/
```

## Boundary

This repo is separate from `Y2JCPA/daf-yomi`. Do not modify the Daf Yomi repo for this project.
